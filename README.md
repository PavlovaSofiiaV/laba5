# laba5
Розробити програму для автоматизації роботи ресторану, яка дозволить ефективно керувати меню, обробляти замовлення клієнтів, контролювати роботу персоналу та оптимізувати процес обслуговування з метою підвищення якості сервісу і зменшення часу обробки замовлень.
Учасники: Шишко Руслана, Павлова Софія
